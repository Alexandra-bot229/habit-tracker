# 📊 Трекер привычек

Веб-приложение для отслеживания ежедневных привычек. Помогает формировать полезные привычки и следить за прогрессом.

## ✨ Функции

- Регистрация и вход (у каждого пользователя свои привычки)
- Добавление привычек (название, описание, цель)
- Отметка выполнения привычки за сегодня
- Визуальный прогресс (проценты + progress bar)
- История выполнения за последние 7 дней
- Поиск по названию привычки
- Удаление привычек
