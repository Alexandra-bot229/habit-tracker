# Инструкция для пользователя «Трекер привычек»

## Регистрация и вход
1. Открыть http://127.0.0.1:5000
2. Зарегистрироваться (логин/пароль)
3. Войти

## Добавление привычки
- Ввести название, описание, цель (раз в день)
- Нажать «Добавить»

## Отметка выполнения
- Нажать «Выполнить» — привычка отмечается за сегодня
- Прогресс-бар обновляется

## История
- Нажать «История» — увидеть выполнение за 7 дней

## Поиск
- Ввести текст в поле поиска → «Найти»

## Удаление
- Нажать «Удалить» → подтвердить
